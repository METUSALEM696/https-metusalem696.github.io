# Arduino / ESP32 Custom Projects

I build custom IoT and automation projects using Arduino and ESP32.

## What I can build
- Sensors + data logging (I2C/SPI/UART)
- WiFi / MQTT / HTTP integrations
- Displays (LCD, OLED, TFT)
- Relay / motor / pump control
- Non-blocking firmware (timers, state machines)

## Deliverables
- Clean, tested code (Arduino IDE / PlatformIO)
- Wiring diagram (annotated)
- Setup instructions + basic support
